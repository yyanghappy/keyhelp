# KeyHelp - 安卓自动化工具

> 个人自用的安卓轻量化自动化工具，支持脚本录制、定时执行和流程控制。

## 特性

- **无障碍服务**：基于Android AccessibilityService实现真实的触控监听和模拟
- **脚本录制**：录制点击、滑动、长按等操作
- **脚本执行**：精确回放录制的操作
- **定时任务**：设置定时自动执行脚本
- **流程控制**：支持条件和循环逻辑
- **极简设计**：无冗余功能，专注核心需求

## 快速开始

### 环境要求

- Flutter 3.24+
- Android SDK 21+ (支持安卓5.0+)
- macOS / Linux / Windows

### 安装

1. 安装Flutter SDK

```bash
./scripts/install_flutter.sh
```

2. 运行应用

```bash
./scripts/run.sh
```

## 使用指南

### 1. 启用无障碍服务

- 首次启动会提示"无障碍服务未启用"
- 点击右上角警告图标或"去设置"按钮
- 在系统设置中启用"KeyHelp"无障碍服务
- 启用后图标变为绿色即可使用

### 2. 脚本录制

- 打开"脚本录制"页面
- 点击"开始"按钮
- 在屏幕上执行操作
- 点击"停止"并保存

### 3. 脚本执行

- 在"脚本列表"中查看所有脚本
- 点击播放图标执行

### 4. 定时任务

- 在"定时任务"页面管理
- 设置执行时间并启用
