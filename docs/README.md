# KeyHelp 安卓自动化工具

## 项目概述

个人自用的安卓轻量化自动化工具，支持脚本录制、定时执行和流程控制。

### 核心特性

- 操作极简：原生安卓体验，无冗余功能
- 功能聚焦：脚本录制、定时执行、触控模拟
- 系统适配：支持安卓12-15，屏幕自适应
- 极致轻量：无商用功能，纯个人工具

### 技术栈

- **框架**: Flutter 3.24+
- **语言**: Dart 3.5+
- **UI框架**: Material Design 3
- **状态管理**: Riverpod 2.5+
- **持久化**: Hive 2.2+（轻量级本地存储）
- **权限管理**: permission_handler 11.3+
- **无障碍服务**: 用于触控模拟和脚本执行
